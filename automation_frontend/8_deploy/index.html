grunt-shipit

  • Выкачивает из репо
  • Позволяет собрать ресурсы по эвентам (init, fetch etc)
  • Выкладывает на сервер
  • Отправляет команды
  • Держит старые релизы, на которые можно откатить

Можно проще

grunt-ssh 
        options: {host, user, pwd}
        command: 'git pull --rebase && npm i && grunt build'
